## How this vault works

- Agents are the only writers. All changes go through the MCP server, which enforces permissions, locking, and audit.
- Everything an agent writes starts as `status: unverified` until promoted to `canonical` by the admin agent on the owner's confirmation.
- One note per entity. The master list lives in [[Entity Index]].
